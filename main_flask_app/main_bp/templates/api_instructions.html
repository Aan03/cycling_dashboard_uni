Cycling Parking API Instructions:

API Routes:
  • http://127.0.0.1:5000/api/reports
    • Get all reports.

  • http://127.0.0.1:5000/api/reports/borough/[Enter borough name]
    • Get all reports for a specific borough.
    • Example: Enfield

  • http://127.0.0.1:5000/api/reports/rack/[Enter rack ID]
    • Get all reports for a specific bike rack.
    • Example: RWG148175

On this website, the "Download Data" page has been designed to use the API to generate a downloadable CSV file.
